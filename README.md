# RoomLearning
Practice of using Room library. Adding, updating and delete elements of Recycler View which are located in Room database.

<h3>Used technologies:</h1>
<ul> 
  <li>ViewBinding</li>
  <li>Room</li>
  <li>Coroutines</li>
  <li>Android Navigation component</li>
</ul>  

<h3>Project structure:</h1>
<ul> 
<li>1 activity</li> 
<li>3 fragments</li> 
</ul>
